# Frontend Integration Guide

## Base URL

```
Production: https://api.vkus.online/api/v1
Development: http://localhost:8000/api/v1
```

## Required Headers

| Header | When | Description |
|--------|------|-------------|
| `Authorization` | Customer endpoints | `Bearer <access_token>` |
| `X-Guest-Session-ID` | Guest endpoints | UUID generated by frontend |
| `Idempotency-Key` | Checkout, payment | UUID to prevent duplicates |
| `X-Request-ID` | Optional | Request tracing ID |
| `Content-Type` | POST/PUT/PATCH | `application/json` |

## Response Envelope

All responses follow this structure:

```typescript
interface ApiResponse<T> {
  ok: boolean;
  data?: T;
  error?: {
    code: string;
    message: string;
    details?: Record<string, any>;
  };
  request_id: string | null;
}
```

## Error Codes

| Code | HTTP | Description |
|------|------|-------------|
| `VALIDATION_ERROR` | 422 | Invalid request data |
| `AUTH_ERROR` | 401 | Missing or invalid auth |
| `FORBIDDEN` | 403 | Access denied |
| `NOT_FOUND` | 404 | Resource not found |
| `CONFLICT` | 409 | Duplicate or state conflict |
| `RATE_LIMIT` | 429 | Too many requests |
| `PROVIDER_*_ERROR` | 502 | External provider failure |
| `INTERNAL_ERROR` | 500 | Server error |

## Auth Flow

```typescript
// 1. Register
const { data } = await api.post('/auth/register', {
  email, password, first_name, last_name
});
// data: { access_token, refresh_token, token_type, expires_in }

// 2. Login
const { data } = await api.post('/auth/login', { email, password });

// 3. Store tokens
localStorage.setItem('access_token', data.access_token);
localStorage.setItem('refresh_token', data.refresh_token);

// 4. Use access token
api.defaults.headers['Authorization'] = `Bearer ${data.access_token}`;

// 5. Refresh when expired
const { data } = await api.post('/auth/refresh', {
  refresh_token: localStorage.getItem('refresh_token')
});
// Update both tokens

// 6. Logout
await api.post('/auth/logout', {
  refresh_token: localStorage.getItem('refresh_token')
});
```

## Guest Flow

```typescript
// 1. Generate guest session ID on first visit
const guestId = localStorage.getItem('guest_session_id') || crypto.randomUUID();
localStorage.setItem('guest_session_id', guestId);

// 2. Bootstrap session
await api.post('/guest/session/bootstrap', {
  guest_session_id: guestId
});

// 3. Include in all guest requests
api.defaults.headers['X-Guest-Session-ID'] = guestId;

// 4. On login, merge guest session
await api.post('/me/merge-guest-session', null, {
  headers: { 'X-Guest-Session-ID': guestId }
});
```

## Checkout Flow

```typescript
// 1. Set cart items
await api.put('/guest/cart/items', [
  { product_sku: 'JG-100', quantity: 2 },
  { product_sku: 'BS-250', quantity: 1 }
]);

// 2. Get delivery estimate
const estimate = await api.post('/delivery/5post/estimate', { ... });

// 3. Get quote
const quote = await api.post('/checkout/quote', {
  items: [{ sku: 'JG-100', quantity: 2 }],
  delivery_provider: '5post',
  city: 'Moscow'
});

// 4. Create order (idempotent!)
const idempotencyKey = crypto.randomUUID();
const order = await api.post('/guest/checkout/create-order', {
  items: [...],
  customer_email: 'user@example.com',
  customer_phone: '+79001234567',
  customer_name: 'Name',
  delivery_provider: '5post',
  delivery_city: 'Moscow',
  pickup_point_id: '...',
  idempotency_key: idempotencyKey
});
// order.data: { order_number, status, total, guest_order_token }

// 5. Create payment
const payment = await api.post(
  `/guest/orders/${order.data.order_number}/payments/yookassa/create`,
  { idempotency_key: crypto.randomUUID() }
);
// payment.data: { payment_id, confirmation_url, status }

// 6. Redirect to payment
window.location.href = payment.data.confirmation_url;

// 7. On return, check status
const status = await api.get(`/guest/orders/${orderNumber}/status`);
```

## TypeScript API Adapter

```typescript
// src/services/api.ts
import axios from 'axios';

const api = axios.create({
  baseURL: import.meta.env.VITE_API_URL || 'http://localhost:8000/api/v1',
  timeout: 30000,
  headers: { 'Content-Type': 'application/json' },
});

// Request interceptor: add auth headers
api.interceptors.request.use((config) => {
  const token = localStorage.getItem('access_token');
  if (token) config.headers.Authorization = `Bearer ${token}`;

  const guestId = localStorage.getItem('guest_session_id');
  if (guestId) config.headers['X-Guest-Session-ID'] = guestId;

  return config;
});

// Response interceptor: handle token refresh
api.interceptors.response.use(
  (response) => response.data,
  async (error) => {
    if (error.response?.status === 401) {
      const refreshToken = localStorage.getItem('refresh_token');
      if (refreshToken) {
        try {
          const { data } = await axios.post(
            `${api.defaults.baseURL}/auth/refresh`,
            { refresh_token: refreshToken }
          );
          localStorage.setItem('access_token', data.data.access_token);
          localStorage.setItem('refresh_token', data.data.refresh_token);
          error.config.headers.Authorization = `Bearer ${data.data.access_token}`;
          return api.request(error.config);
        } catch {
          localStorage.removeItem('access_token');
          localStorage.removeItem('refresh_token');
        }
      }
    }
    return Promise.reject(error);
  }
);

export default api;
```

## Pagination

Paginated endpoints return:

```json
{
  "ok": true,
  "data": {
    "items": [...],
    "total": 42,
    "page": 1,
    "per_page": 20
  }
}
```

Query params: `?page=1&per_page=20`
